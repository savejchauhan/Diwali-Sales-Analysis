 Diwali Sales Analysis

 Project Overview
This project analyzes Diwali sales data using Python libraries like Pandas and Matplotlib.

Key Insights
- Identified top states contributing highest sales
- Analyzed gender-wise and age group purchasing behavior
- Found most profitable product categories
- Visualized sales trends using charts

Tools & Technologies
- Python
- Pandas
- Matplotlib

Dataset
Diwali sales dataset containing customer and purchase details

Output
Charts and insights generated using Python
